Question and Answer

1. What is DOM and what is the purpose of DOM?

Answer: The full name of DOM is Document Object Model and it gives us a html document tree. Using DOM we can manipulate website content. We set website interactivity using DOM

2. How will you select HTML elements using DOM? Name the DOM methods.

Answer: We can use id, class in HTML element or we can directly select element using tag of html elements through DOM. There are so many DOM selectors. DOM selector methods are:

1. getElementById()

2. getElementsByClassName()

3. getElementsByTagName()

4. querySelector()

5. querySelector()

3. What is event bubble?

Answer:Event Bubbling is a concept in Document Object Model. It receives an event and then continues to bubble with its parent nodes and works until it comes to the main element of that event

4. What is a callback function and why will you use it?

Answer:A callback function is a function passed as an argument to another function that gets called at some point during the execution of the main function. It's commonly used for event-driven programming and allows for additional processing or value return based on the callback function's execution.
