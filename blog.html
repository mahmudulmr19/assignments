Question: Discuss the scope of var, let, and const

Answer: In JavaScript, variables declared using `var`, `let`, and `const` have different scopes. The scope of a variable defines where it can be accessed or used within your code.

Question: What do you mean by REST API?

Answer: REST API, which stands for Representational State Transfer Application Programming Interface, is an architectural style for designing networked applications.
